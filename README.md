## âœ¨ Key Features

* **GSAP ScrollTrigger:** Implements a horizontal scrolling text section ("You Think, I Code") that pins and scrubs based on scroll position.
* **Elastic SVG String:** An interactive guitar-string effect using SVG paths and quadratic Bezier curves (`Q`) that snaps back elastically when hovered.
* **Custom Cursor & Blending:** A "spotlight" cursor effect in the footer using CSS `mix-blend-mode: difference`.
* **Floating 3D Elements:** Hero section features floating technology icons with independent animation delays.
* **Project Hover Reveal:** Dynamic image preview that follows the mouse cursor when hovering over project names.
* **Real-time Clock:** Displays the current user time in IST.
